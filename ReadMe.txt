To open the quiz, using Visual Studio Codes Live Server test, open all the files in VSCode.
Navigate to Index.html, right click the empty space, and then with the live server extension installed, click "Open with Live Server".
From there, the quiz will be available to take.

Questions will be scored for right and wrong answers.
Users can save their name at the end, and the last 5 highest scores can be viewed from the index.
